# Synthetic Data and Machine Learning for Geothermal Reservoir Temperature Estimation in Western Anatolia

This repository contains the code and notebooks used in the study:

Advancing Geothermal Reservoir Temperature Prediction in  The Western Anatolia (Türkiye) Using Synthetic Hydrogeochemical Data from Rank–Gaussian Copulas

Füsun S. Tut Haklıdır, Mehmet Haklıdır

1 Istanbul Bilgi University, Department of Energy Systems Engineering, Eyüp Istanbul-Türkiye
2 TÜBİTAK BİLGEM, Gebze Kocaeli-Türkiye


The code implements:
- A **rank–Gaussian copula (RGC)** synthetic data generator tailored to routine hydrogeochemical variables in Western Anatolia.
- A **DNN ensemble** with pre-training on RGC-synthetic data and fine-tuning on real data.
- **NGBoost** models trained on real-only and RGC-augmented datasets.

The notebooks reproduce the main quantitative results reported in Sections 3–5 of the manuscript, including:
- Marginal and multivariate diagnostics for synthetic data (Tables 3–4, Figures 3–7),
- DNN ensemble performance (Table 6a, Figure 11),
- NGBoost comparison with and without RGC augmentation (Table 6b, Figures 12–13).
